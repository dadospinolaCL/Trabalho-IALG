# Trabalho final do primeiro período da faculdade


## Objetivo
- Criar uma struct com o tema música com os campos: nome (string), artista (string), duração (string), ano (int), views (long long), media_views (double), descrição (string)
  
## Funções:
    - Exibir dados (músicas)
          - Exibir todos os dados
          - Exibir uma parte escolhida com dois índices pelo usuário
    - Busca
          - Busca por nome da música
          - Busca por nome do artista
          - Busca por uma parte da música (que esteja no campo descrição)
    - Ordenação
          - Por nome (ordem alfabética)
          - Views (decrescente)
          - Média de views (decrescente)
    - Salvar
          - Após alterações feitas, é possível salvá-las sobrescrevendo o banco de dados
